# System Evolution

This project did not start as a complete system.

It began with web log–based anomaly detection,
and gradually evolved into a structure that includes
risk calculation, root cause analysis, action generation,
ML-based prediction, AI-driven explanation,
and operational observability.

This document serves as an index that outlines
the evolution of the system across different stages.

---

## Evolution Path

v0.1 → Data Reliability Foundation  
v0.2 → Stream Signal, ML, AI Chain  
v0.3 → Unified Reliability Architecture  
v0.4 → Domain and Decision Expansion  
v0.5 → Financial Domain and State-aware Reliability
